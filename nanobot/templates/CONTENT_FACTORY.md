# Content Factory

Use this file to guide recurring content generation in this workspace.

## Account Positioning
- Main positioning: finance and AI content with image-card-first delivery
- Audience: users interested in investing, funds, wealth management, and practical AI tools
- Primary goals:
  - Grow followers
  - Build trust through recognizable scenes, not lectures
  - Create future monetization paths through templates, tools, consulting, or paid research

## Topic Pillars
1. Funds and wealth management pain points
2. Retail investor behavior labels and personas
3. AI tools and products with direct everyday value
4. Finance meme-like small scenes that imply bigger mistakes

## Content Filters
- Prefer tiny relatable scenes, strong labels, and self-recognition
- Avoid empty macro commentary with no action value
- Avoid direct buy/sell instructions or return promises
- Prefer topics that can become a repeatable image-card series

## Platform Notes

### Xiaohongshu
- Image-first
- One card, one joke, one role
- Optimize for self-recognition, saves, comments, and shares
- Prefer punchlines over long explanations

### Weibo
- Prioritize labels, punchlines, and grouped repostable cards
- Keep the text compact and direct

### Future Platforms
- Douyin: keep adaptation notes only
- X: keep adaptation notes only

## Default IP
- Mascot: TradingCat
- Role system: same cat, different costumes, props, and expressions
- Tone: internet-friend roast, not teacher mode

## Standard Output
Every content pack should include:
- One-line hook
- Xiaohongshu cover copy
- Per-card overlay text
- Per-card TradingCat role description
- Per-card English image prompt
- Weibo adaptation
- Douyin adaptation notes
- X adaptation notes
- CTA / comment bait
- 3 follow-up topic ideas

## Image Workflow
- Stage prompts first; do not generate immediately
- Confirm each image one by one
- Commands:
  - `/image-confirm`
  - `/image-edit <feedback>`
  - `/image-skip`
- Save outputs under `content_queue/YYYY-MM-DD-slug.md`

## Working Style
- If no topic is specified, propose 3 candidates first
- If a topic is selected, produce the full pack directly
- Prefer image-card-first output over long article output
